# 💼 JobMatch

A Full Stack Job Portal built with the MERN Stack.

## 🚀 Live Demo

https://jobmatch-cfxi.vercel.app/

## ✨ Features

- User Authentication
- Job Posting
- Job Applications
- Search & Filtering
- Responsive Design
- REST API
- MongoDB Database

## 🛠 Tech Stack

### Frontend
- React.js
- HTML
- CSS
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## 📦 Installation

```bash
git clone https://github.com/israoun11/jobmatch.git

cd jobmatch

npm install

npm run dev
```
